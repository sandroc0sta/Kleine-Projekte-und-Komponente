PERSONEN-ZÄHLER (Train Entry Counter)
=====================================

Diese einfache JavaScript-Anwendung zählt, wie viele Personen in einen Zug eingestiegen sind.
Die Anwendung kann problemlos angepasst und für andere Zählszenarien verwendet werden.

Funktionen:
- Erhöhen der Zählung (increment)
- Verringern der Zählung (decrement)
- Zurücksetzen der Zählung (reset)
- Speichern des aktuellen Zählstands (save)

Ideal für kleine Zählszenarien und als Übungsprojekt zum Einstieg in JavaScript.



PEOPLE COUNTER (Train Entry Counter)
====================================

This simple JavaScript app counts how many people enter a train.
It can be easily adapted for other counting purposes.

Features:
- Increment the count
- Decrement the count
- Reset the count
- Save the current count

Perfect for small counting tasks and a beginner-friendly JavaScript project.
