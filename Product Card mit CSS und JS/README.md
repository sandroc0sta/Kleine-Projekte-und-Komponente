# Kleine Produktkarten-Komponente

Eine kleine Produktkarten-Komponente mit CSS-Animationen unter Verwendung von **Transitions**, **Keyframe-Animationen** und **Flexbox**.  
Zusätzlich enthält sie eine JavaScript-Funktion, um die Karte beim Klicken hervorzuheben (Highlighting).

## Verwendete Technologien

- CSS Transitions (für sanfte Übergänge)
- CSS Keyframe-Animationen (für den Puls-Effekt)
- Flexbox (für das Layout)
- JavaScript (für das Umschalten der Hervorhebung beim Klick)

# Small Product Card Component

A small product card component with CSS animations using **transitions**, **keyframe animations**, and **flexbox**.  
Additionally, it includes a JavaScript function to highlight the card when clicked.

## Technologies Used

- CSS transitions (for smooth effects)
- CSS keyframe animations (for the pulse effect)
- Flexbox (for layout)
- JavaScript (for toggling highlight on click)
